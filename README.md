# NZ Math Hub

A website for teaching mathematics to New Zealand secondary school students, organized by Year Level with appropriate content structure for junior and senior levels.

## Overview

NZ Math Hub provides mathematics resources organized by:
- **Year 9-10 (Junior Levels)**: Content organized by mathematics strands (Number, Algebra, Geometry, Measurement, Statistics, Probability)
- **Year 11-13 (Senior Levels)**: Content organized by NCEA achievement standards (Levels 1-3)

## Structure

The website consists of:
- `index.html` - Homepage with Year Level navigation
- `year9.html` - Year 9 mathematics strands
- `year10.html` - Year 10 mathematics strands
- `year11.html` - NCEA Level 1 standards
- `year12.html` - NCEA Level 2 standards
- `year13.html` - NCEA Level 3 standards
- `styles.css` - Responsive styling for all pages

## Features

- Clean, modern design with intuitive navigation
- Responsive layout that works on desktop, tablet, and mobile devices
- Clear visual distinction between junior and senior year levels
- Organized by mathematics strands for Years 9-10
- Organized by NCEA achievement standards for Years 11-13
- Easy to extend with additional content and resources

## Usage

To view the website locally:
1. Clone this repository
2. Open `index.html` in a web browser

Alternatively, you can serve it using a local web server:
```bash
python3 -m http.server 8000
```
Then navigate to `http://localhost:8000` in your web browser.
