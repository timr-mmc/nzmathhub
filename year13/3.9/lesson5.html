1

Using the Regression Equation

#

Once we have the regression equation, we can use it to make predictions. The equation $y = a + bx$ allows us to:

  • Predict y for a given value of x
  • Estimate x for a given value of y (by rearranging)
  • Understand how changes in x affect y

Example: Study Time and Test Scores

Regression equation: $\text{Score} = 45 + 5(\text{Hours})$

Predict the test score for 6 hours of study:

$$\text{Score} = 45 + 5(6) = 45 + 30 = 75\%$$

Answer: We predict a test score of 75% for a student who studies 6 hours.

How many hours needed for a score of 80%?

$$80 = 45 + 5(\text{Hours})$$ $$35 = 5(\text{Hours})$$ $$\text{Hours} = 7$$

Answer: A student would need to study approximately 7 hours to achieve 80%.

💡 Remember: Always interpret predictions in context and include appropriate units.
2

Interpolation

#
Definition: Interpolation is making predictions within the range of the observed data. These predictions are generally reliable if the linear model fits well.

When to Use Interpolation

  • The x-value is within the range of observed data
  • The relationship appears linear in that range
  • There are no unusual features (outliers, gaps) near the prediction point

Example: Temperature and Ice Cream Sales

Data collected for temperatures between 15°C and 35°C.

Regression equation: $\text{Sales} = -200 + 15(\text{Temp})$

Predict sales when temperature is 25°C:

$$\text{Sales} = -200 + 15(25) = -200 + 375 = \$175$$

Why This is Interpolation: 25°C is within the observed range (15°C to 35°C), so this prediction is reasonably reliable.

✓ Interpolation is Generally Reliable: We're predicting within known conditions, so the linear pattern is likely to hold.

Reliability of Interpolation

Interpolated predictions are more reliable when:

  • The correlation is strong (|r| close to 1)
  • R² is high (close to 1)
  • The scatter plot shows a clear linear pattern
  • There are no outliers near the prediction point
  • The sample size is reasonable
3

Extrapolation

#
Definition: Extrapolation is making predictions outside the range of observed data. These predictions are less reliable and should be made with caution.

Dangers of Extrapolation

  • The linear relationship might not continue outside the observed range
  • Other factors might become important at extreme values
  • The model may break down completely
  • Predictions can be wildly inaccurate

Example: Temperature and Ice Cream Sales (Continued)

Using the same equation: $\text{Sales} = -200 + 15(\text{Temp})$

Predict sales when temperature is 50°C:

$$\text{Sales} = -200 + 15(50) = -200 + 750 = \$550$$

Why This is Problematic:

  1. 50°C is well outside the observed range (15°C - 35°C)
  2. At 50°C, other factors might matter (people staying indoors, health warnings)
  3. The linear relationship might not hold at extreme temperatures
  4. This prediction should be treated with skepticism
⚠️ Warning: Always state when you're extrapolating and acknowledge the increased uncertainty!

Another Example: Age and Height

Data on children aged 5-15 years: $\text{Height} = 80 + 6(\text{Age})$

Predicting height at age 40:

$$\text{Height} = 80 + 6(40) = 320 \text{ cm}$$

This is Absurd! 320 cm is over 10 feet tall. This shows why extrapolation can fail - people stop growing, so the linear model breaks down outside the age range where it was developed.

When Extrapolation Might Be Acceptable

Extrapolation can be cautiously used when:

  • Extrapolating only slightly beyond the data range
  • There's strong theoretical reason to believe the relationship continues
  • The correlation is very strong
  • You clearly state the prediction's limitations

Writing About Predictions

Good Practice:

Interpolation: "Based on the regression model, we predict [predicted value] for [x value]. This prediction is reasonably reliable as it falls within the range of observed data and the model shows a [strong/moderate] fit with R² = [value]."

Extrapolation: "Based on the regression model, we predict [predicted value] for [x value]. However, this is an extrapolation beyond the observed data range, so this prediction should be treated with caution as the linear relationship may not hold outside the observed range."
