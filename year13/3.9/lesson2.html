Creating Scatter Plots

#
Definition: A scatter plot (or scatter graph) is a graph that displays bivariate data as points on a coordinate plane.

Steps to Create a Scatter Plot

  1. Identify Variables: Determine which is explanatory (x-axis) and response (y-axis)
  2. Choose Scale: Select appropriate scales that fit all data points
  3. Label Axes: Clearly label both axes with variable names and units
  4. Plot Points: Mark each data pair as a point on the graph
  5. Add Title: Give your scatter plot a descriptive title

Axis Conventions

  • Horizontal axis (x): Explanatory/independent variable
  • Vertical axis (y): Response/dependent variable
  • Both axes should start at zero when appropriate
  • Use equal intervals on each axis
💡 Tip: Always plot the explanatory variable on the x-axis. This helps maintain consistency when analyzing and making predictions.

Example Data: Study Time vs Test Score

Student Study Time (hours) Test Score (%)
A 2 55
B 4 68
C 6 75
D 3 62
E 8 85

This data would show a positive relationship when plotted.

2

Describing Scatter Plots

#

When describing a scatter plot, use the DFSU framework:

D - Direction

  • Positive: As x increases, y increases
  • Negative: As x increases, y decreases
  • No apparent relationship: No clear pattern

F - Form

  • Linear: Points follow a straight-line pattern
  • Curved: Points follow a curved pattern (not in Level 3)

S - Strength

  • Strong: Points cluster tightly around a line/pattern
  • Moderate: Points show general trend but with scatter
  • Weak: Points show little adherence to pattern

U - Unusual Features

  • Outliers: Points that don't fit the general pattern
  • Clusters: Groups of points separated from others
  • Gaps: Areas with no data points
Example Description:
"The scatter plot shows a strong, positive, linear relationship between study time and test scores. As study time increases, test scores tend to increase. There are no obvious outliers or unusual features."
3

Interpreting Scatter Plots

#

What Scatter Plots Tell Us

1. Association between variables: Whether variables are related

2. Trend direction: Whether the relationship is positive or negative

3. Relationship strength: How closely variables are related

4. Unusual observations: Data points that don't fit the pattern

What Scatter Plots DON'T Tell Us

  • Causation: Just because variables are related doesn't mean one causes the other
  • Exact predictions: We can only estimate, not determine exact values
  • Relationships outside the data range: Patterns may not continue beyond observed data
⚠️ Warning: Correlation ≠ Causation!

Example: Ice cream sales and drowning deaths are correlated (both increase in summer), but ice cream doesn't cause drownings. The lurking variable is temperature/season.

Context is Critical

Always interpret scatter plots in context:

  • What do the variables represent?
  • What are the units of measurement?
  • What population does the data represent?
  • Are there any limitations to the data?
