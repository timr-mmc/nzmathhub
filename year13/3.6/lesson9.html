Finding Maximum and Minimum Values

Optimisation uses differentiation to find the best solution - the maximum profit, minimum cost, largest area, etc. These are practical applications of stationary points.

Optimisation Strategy

  1. Identify what to optimise: What are you maximising/minimising?
  2. Define variables: Choose appropriate letters
  3. Write formula: Express quantity in terms of one variable
  4. Use constraints: Use given information to eliminate extra variables
  5. Differentiate and solve: Find stationary points
  6. Verify it's a max/min: Use second derivative test
  7. Answer the question: State optimal value clearly

Common Optimisation Types

  • Area/Perimeter: Maximise area with fixed perimeter (or vice versa)
  • Volume/Surface Area: Maximise volume with fixed surface area
  • Cost: Minimise costs given constraints
  • Distance: Find closest point or shortest path

Worked Examples

Example 1: Maximum Area with Fixed Perimeter

A rectangle has perimeter 40 m. Find dimensions that give maximum area.

1 Variables: Let length = $x$, width = $y$. Maximise Area = $A = xy$
2 Constraint: Perimeter: $2x + 2y = 40$, so $y = 20 - x$
3 Express A in terms of x: $A = x(20 - x) = 20x - x^2$
4 Differentiate: $\frac{dA}{dx} = 20 - 2x$
5 Find stationary point: $20 - 2x = 0$, so $x = 10$ m
6 Verify maximum: $\frac{d^2A}{dx^2} = -2 < 0$ → maximum ✓
7 Answer: $x = 10$ m, $y = 10$ m. Square gives maximum area of 100 m²

Example 2: Minimum Cost Fence

A rectangular field of area 1800 m² is to be fenced. Fencing costs $20/m for three sides and $40/m for the fourth side. Find dimensions for minimum cost.

1 Variables: Length $x$, width $y$. Area: $xy = 1800$
2 Cost function: $C = 40x + 20x + 20y + 20y = 60x + 40y$
3 Eliminate y: $y = \frac{1800}{x}$, so $C = 60x + 40(\frac{1800}{x}) = 60x + \frac{72000}{x}$
4 Differentiate: $\frac{dC}{dx} = 60 - \frac{72000}{x^2}$
5 Solve: $60 - \frac{72000}{x^2} = 0$
$60x^2 = 72000$
$x^2 = 1200$, so $x = 20\sqrt{3} \approx 34.6$ m
6 Find y: $y = \frac{1800}{20\sqrt{3}} = 30\sqrt{3} \approx 52$ m
7 Verify: $\frac{d^2C}{dx^2} = \frac{144000}{x^3} > 0$ → minimum ✓

Example 3: Maximum Volume Box

A box with square base and open top must have volume 32 m³. Find dimensions that minimise surface area.

1 Variables: Base side $x$, height $h$. Volume: $V = x^2h = 32$
2 Surface area: Base + 4 sides = $x^2 + 4xh$
3 Eliminate h: $h = \frac{32}{x^2}$, so $S = x^2 + 4x(\frac{32}{x^2}) = x^2 + \frac{128}{x}$
4 Differentiate: $\frac{dS}{dx} = 2x - \frac{128}{x^2}$
5 Solve: $2x - \frac{128}{x^2} = 0$
$2x^3 = 128$
$x^3 = 64$, so $x = 4$ m
6 Find h: $h = \frac{32}{16} = 2$ m
7 Answer: Base 4 m × 4 m, height 2 m minimises surface area

Practice: Minimal Change

Set 1: Rectangle - varying perimeter

Maximise area with different fixed perimeters

a) Rectangle with perimeter 20 m. Find dimensions for maximum area.
$2x + 2y = 20$, so $y = 10 - x$
$A = x(10 - x) = 10x - x^2$
$\frac{dA}{dx} = 10 - 2x = 0 \Rightarrow x = 5$
Maximum area when $x = y = 5$ m (square)
b) Rectangle with perimeter 60 m. Find dimensions for maximum area.
$y = 30 - x$
$A = x(30 - x) = 30x - x^2$
$\frac{dA}{dx} = 30 - 2x = 0 \Rightarrow x = 15$
Maximum area when $x = y = 15$ m
c) Rectangle with perimeter 100 m. Find dimensions for maximum area.
$y = 50 - x$
$A = x(50 - x) = 50x - x^2$
$\frac{dA}{dx} = 50 - 2x = 0 \Rightarrow x = 25$
Maximum area when $x = y = 25$ m

Set 2: Rectangle - varying area

Minimise perimeter with different fixed areas

a) Rectangle with area 100 m². Find dimensions for minimum perimeter.
$xy = 100$, so $y = \frac{100}{x}$
$P = 2x + 2y = 2x + \frac{200}{x}$
$\frac{dP}{dx} = 2 - \frac{200}{x^2} = 0 \Rightarrow x^2 = 100$, $x = 10$
Minimum perimeter when $x = y = 10$ m
b) Rectangle with area 400 m². Find dimensions for minimum perimeter.
$P = 2x + \frac{800}{x}$
$\frac{dP}{dx} = 2 - \frac{800}{x^2} = 0 \Rightarrow x = 20$
Minimum when $x = y = 20$ m
c) Rectangle with area 900 m². Find dimensions for minimum perimeter.
$P = 2x + \frac{1800}{x}$
$\frac{dP}{dx} = 2 - \frac{1800}{x^2} = 0 \Rightarrow x = 30$
Minimum when $x = y = 30$ m

Set 3: Box problems - varying volume

Open-top box with square base, minimise surface area

a) Volume 64 m³. Find dimensions to minimise surface area.
$x^2h = 64$, so $h = \frac{64}{x^2}$
$S = x^2 + 4xh = x^2 + \frac{256}{x}$
$\frac{dS}{dx} = 2x - \frac{256}{x^2} = 0 \Rightarrow x^3 = 128$, $x = 4\sqrt[3]{2} \approx 5.04$ m
$h = \frac{64}{x^2} \approx 2.52$ m
b) Volume 125 m³. Find dimensions to minimise surface area.
$S = x^2 + \frac{500}{x}$
$\frac{dS}{dx} = 2x - \frac{500}{x^2} = 0 \Rightarrow x^3 = 250$, $x = \sqrt[3]{250} \approx 6.3$ m
$h = \frac{125}{x^2} \approx 3.15$ m

Set 4: Mixed problems

a) Sum of two positive numbers is 20. Maximise their product.
Let numbers be $x$ and $20 - x$
Product: $P = x(20 - x) = 20x - x^2$
$\frac{dP}{dx} = 20 - 2x = 0 \Rightarrow x = 10$
Maximum product when both numbers are 10
b) A wire 120 cm long is cut into two pieces. One forms a square, the other a circle. Where to cut to minimise total area?
Let $x$ = length for square, $120 - x$ for circle
Square side: $\frac{x}{4}$, area: $\frac{x^2}{16}$
Circle: $2\pi r = 120 - x$, so $r = \frac{120 - x}{2\pi}$, area: $\frac{(120-x)^2}{4\pi}$
Total: $A = \frac{x^2}{16} + \frac{(120-x)^2}{4\pi}$
$\frac{dA}{dx} = \frac{x}{8} - \frac{120-x}{2\pi} = 0$
$x \approx 52.7$ cm for square

Key Points to Remember

  • ✓ Identify what to optimise and what constraints exist
  • ✓ Express in one variable using constraints to eliminate others
  • ✓ Differentiate and find stationary points
  • ✓ Always verify using second derivative test
  • ✓ Check answer makes practical sense (positive dimensions, etc.)
  • ✓ For many problems, square beats rectangle (equal dimensions optimal)
