The Power Rule: Foundation of Differentiation

The power rule is the most fundamental differentiation technique. Master this, and you'll be able to differentiate any polynomial function quickly and accurately.

The Power Rule

If $f(x) = x^n$, then:

$$f'(x) = nx^{n-1}$$

In words: "Multiply by the power, then reduce the power by 1"

Worked Examples

Let's see how the power rule works with clear, step-by-step examples.

Example 1: Basic Power

Differentiate: $f(x) = x^5$
Step 1: Identify the power ▼
$n = 5$
Step 2: Multiply by the power ▼
$5 \times x^?$
Step 3: Reduce the power by 1 ▼
$5 - 1 = 4$
✓ Answer ▼
$f'(x) = 5x^4$

Example 2: With Coefficient

Differentiate: $f(x) = 3x^4$
Step 1: Note the coefficient ▼
The 3 stays with us
Step 2: Apply power rule ▼
$3 \times 4x^{4-1}$
Step 3: Simplify ▼
$3 \times 4 = 12$
✓ Answer ▼
$f'(x) = 12x^3$

Example 3: Negative Power

Differentiate: $f(x) = x^{-2}$
Step 1: The power is negative ▼
$n = -2$
Step 2: Multiply by the power ▼
$-2 \times x^?$
Step 3: Reduce power by 1 ▼
$-2 - 1 = -3$
✓ Answer ▼
$f'(x) = -2x^{-3}$

Example 4: Fractional Power

Differentiate: $f(x) = x^{1/2}$ (which is $\sqrt{x}$)
Step 1: The power is a fraction ▼
$n = \frac{1}{2}$
Step 2: Multiply by the power ▼
$\frac{1}{2} \times x^?$
Step 3: Reduce power by 1 ▼
$\frac{1}{2} - 1 = -\frac{1}{2}$
✓ Answer ▼
$f'(x) = \frac{1}{2}x^{-1/2}$ or $\frac{1}{2\sqrt{x}}$

Practice: Minimal Change Problems

These problems change one thing at a time to help you spot the pattern and build confidence.

Set 1: Same form, different powers

Notice: Only the power changes

a) Differentiate: $x^3$
$f'(x) = 3x^2$
b) Differentiate: $x^{4}$
$f'(x) = 4x^3$
c) Differentiate: $x^{6}$
$f'(x) = 6x^5$
d) Differentiate: $x^{7}$
$f'(x) = 7x^6$

Set 2: Adding coefficients

Notice: Now we add a coefficient in front

a) Differentiate: $2x^3$
$f'(x) = 6x^2$
b) Differentiate: $5x^3$
$f'(x) = 15x^2$
c) Differentiate: $4x^5$
$f'(x) = 20x^4$
d) Differentiate: $7x^4$
$f'(x) = 28x^3$

Set 3: Negative powers

Notice: The power is now negative

a) Differentiate: $x^{-1}$
$f'(x) = -x^{-2}$
b) Differentiate: $x^{-3}$
$f'(x) = -3x^{-4}$
c) Differentiate: $2x^{-2}$
$f'(x) = -4x^{-3}$
d) Differentiate: $5x^{-4}$
$f'(x) = -20x^{-5}$

Set 4: Fractional powers

Notice: The power is now a fraction

a) Differentiate: $x^{1/2}$
$f'(x) = \frac{1}{2}x^{-1/2}$
b) Differentiate: $x^{1/3}$
$f'(x) = \frac{1}{3}x^{-2/3}$
c) Differentiate: $x^{3/2}$
$f'(x) = \frac{3}{2}x^{1/2}$
d) Differentiate: $2x^{3/4}$
$f'(x) = \frac{3}{2}x^{-1/4}$

Set 5: Multiple terms (Polynomials)

Notice: Differentiate each term separately

a) Differentiate: $x^3 + x^2$
$f'(x) = 3x^2 + 2x$
b) Differentiate: $2x^4 - 3x^2$
$f'(x) = 8x^3 - 6x$
c) Differentiate: $5x^3 + 4x^2 - 2x + 7$
$f'(x) = 15x^2 + 8x - 2$ (Note: constant 7 becomes 0)
d) Differentiate: $x^5 - 3x^3 + 2x - 8$
$f'(x) = 5x^4 - 9x^2 + 2$

Key Points to Remember

  • ✓ Power rule: Multiply by the power, then reduce the power by 1
  • ✓ The derivative of a constant is 0
  • ✓ The derivative of $x$ is 1 (think: $x^1$ becomes $1 \cdot x^0 = 1$)
  • ✓ Coefficients stay with the term and get multiplied
  • ✓ Negative and fractional powers follow the same rule
  • ✓ Differentiate each term of a polynomial separately
