When Functions Multiply or Divide

When two functions are multiplied or divided, we can't simply differentiate each part separately. We need special rules: the product rule and the quotient rule.

The Product Rule

If $y = u \cdot v$ where both $u$ and $v$ are functions of $x$, then:

$$\frac{dy}{dx} = u\frac{dv}{dx} + v\frac{du}{dx}$$

Or more simply: $y' = uv' + vu'$

In words: "First times derivative of second, plus second times derivative of first"

The Quotient Rule

If $y = \frac{u}{v}$ where both $u$ and $v$ are functions of $x$, then:

$$\frac{dy}{dx} = \frac{v\frac{du}{dx} - u\frac{dv}{dx}}{v^2}$$

Or: $y' = \frac{vu' - uv'}{v^2}$

Memory aid: "Low d-high minus high d-low, over the square of what's below"

Product Rule: Worked Examples

Example 1: Basic Product

Differentiate: $y = x^2(x + 1)$
Step 1: Identify u and v ▼
$u = x^2$ and $v = x + 1$
Step 2: Find derivatives ▼
$u' = 2x$ and $v' = 1$
Step 3: Apply formula ▼
$y' = uv' + vu'$
Step 4: Substitute ▼
$y' = x^2(1) + (x + 1)(2x)$
✓ Answer: Simplify ▼
$y' = x^2 + 2x^2 + 2x = 3x^2 + 2x$

Example 2: Product with Powers

Differentiate: $y = (2x + 3)(x^3)$
Step 1: Identify ▼
$u = 2x + 3$ and $v = x^3$
Step 2: Derivatives ▼
$u' = 2$ and $v' = 3x^2$
Step 3: Apply ▼
$y' = (2x + 3)(3x^2) + (x^3)(2)$
Step 4: Expand ▼
$y' = 6x^3 + 9x^2 + 2x^3$
✓ Answer: Simplify ▼
$y' = 8x^3 + 9x^2$

Quotient Rule: Worked Examples

Example 3: Basic Quotient

Differentiate: $y = \frac{x^2}{x + 1}$
Step 1: Identify ▼
$u = x^2$ (top) and $v = x + 1$ (bottom)
Step 2: Derivatives ▼
$u' = 2x$ and $v' = 1$
Step 3: Apply ▼
$y' = \frac{vu' - uv'}{v^2}$
Step 4: Substitute ▼
$y' = \frac{(x + 1)(2x) - (x^2)(1)}{(x + 1)^2}$
✓ Answer: Simplify ▼
$y' = \frac{2x^2 + 2x - x^2}{(x + 1)^2} = \frac{x^2 + 2x}{(x + 1)^2}$

Example 4: More Complex Quotient

Differentiate: $y = \frac{3x^2 - 1}{2x + 5}$
Step 1: Identify ▼
$u = 3x^2 - 1$ and $v = 2x + 5$
Step 2: Derivatives ▼
$u' = 6x$ and $v' = 2$
Step 3: Apply ▼
$y' = \frac{(2x + 5)(6x) - (3x^2 - 1)(2)}{(2x + 5)^2}$
Step 4: Expand ▼
$y' = \frac{12x^2 + 30x - 6x^2 + 2}{(2x + 5)^2}$
✓ Answer: Simplify ▼
$y' = \frac{6x^2 + 30x + 2}{(2x + 5)^2}$

Product Rule or Quotient Rule?

Here's how to decide which rule to use:

Use Product Rule When:

  • Two functions are multiplied
  • Example: $(x^2)(x + 1)$
  • Example: $(3x)(x^2 - 4)$
  • Look for: $(...)(...) $

Use Quotient Rule When:

  • One function is divided by another
  • Example: $\frac{x^2}{x + 1}$
  • Example: $\frac{2x - 1}{x^3}$
  • Look for: $\frac{...}{...}$

Practice: Product Rule - Minimal Change

Set 1: Changing the first function only

Notice: Only the first function changes

a) $y = x(x + 1)$
$u = x, u' = 1; v = x + 1, v' = 1$
$y' = x(1) + (x + 1)(1) = x + x + 1 = 2x + 1$
b) $y = x^2(x + 1)$
$u = x^2, u' = 2x; v = x + 1, v' = 1$
$y' = x^2(1) + (x + 1)(2x) = x^2 + 2x^2 + 2x = 3x^2 + 2x$
c) $y = x^3(x + 1)$
$u = x^3, u' = 3x^2; v = x + 1, v' = 1$
$y' = x^3(1) + (x + 1)(3x^2) = x^3 + 3x^3 + 3x^2 = 4x^3 + 3x^2$

Set 2: Changing the second function only

Notice: Only the second function changes

a) $y = x^2(x)$
$u = x^2, u' = 2x; v = x, v' = 1$
$y' = x^2(1) + x(2x) = x^2 + 2x^2 = 3x^2$
b) $y = x^2(x + 1)$
$u = x^2, u' = 2x; v = x + 1, v' = 1$
$y' = x^2(1) + (x + 1)(2x) = x^2 + 2x^2 + 2x = 3x^2 + 2x$
c) $y = x^2(x^2 + 1)$
$u = x^2, u' = 2x; v = x^2 + 1, v' = 2x$
$y' = x^2(2x) + (x^2 + 1)(2x) = 2x^3 + 2x^3 + 2x = 4x^3 + 2x$

Practice: Quotient Rule - Minimal Change

Set 3: Changing the numerator only

Notice: Only the top changes

a) $y = \frac{x}{x + 1}$
$u = x, u' = 1; v = x + 1, v' = 1$
$y' = \frac{(x + 1)(1) - x(1)}{(x + 1)^2} = \frac{1}{(x + 1)^2}$
b) $y = \frac{x^2}{x + 1}$
$u = x^2, u' = 2x; v = x + 1, v' = 1$
$y' = \frac{(x + 1)(2x) - x^2(1)}{(x + 1)^2} = \frac{x^2 + 2x}{(x + 1)^2}$
c) $y = \frac{x^3}{x + 1}$
$u = x^3, u' = 3x^2; v = x + 1, v' = 1$
$y' = \frac{(x + 1)(3x^2) - x^3(1)}{(x + 1)^2} = \frac{2x^3 + 3x^2}{(x + 1)^2}$

Set 4: Changing the denominator only

Notice: Only the bottom changes

a) $y = \frac{x^2}{x}$
$u = x^2, u' = 2x; v = x, v' = 1$
$y' = \frac{x(2x) - x^2(1)}{x^2} = \frac{x^2}{x^2} = 1$
b) $y = \frac{x^2}{x + 1}$
$u = x^2, u' = 2x; v = x + 1, v' = 1$
$y' = \frac{(x + 1)(2x) - x^2(1)}{(x + 1)^2} = \frac{x^2 + 2x}{(x + 1)^2}$
c) $y = \frac{x^2}{x^2 + 1}$
$u = x^2, u' = 2x; v = x^2 + 1, v' = 2x$
$y' = \frac{(x^2 + 1)(2x) - x^2(2x)}{(x^2 + 1)^2} = \frac{2x}{(x^2 + 1)^2}$

Key Points to Remember

  • ✓ Product Rule: $y' = uv' + vu'$ (first times d-second plus second times d-first)
  • ✓ Quotient Rule: $y' = \frac{vu' - uv'}{v^2}$ (low d-high minus high d-low, over the square of below)
  • ✓ Always identify $u$ and $v$ first
  • ✓ Find $u'$ and $v'$ before substituting into the formula
  • ✓ Remember to simplify your final answer
  • ✓ For quotients, don't forget to square the denominator!
