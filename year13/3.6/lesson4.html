Combining Differentiation Rules

Real-world problems often require using multiple differentiation rules together. The key is recognizing which rules to apply and in what order.

Strategy for Complex Functions

  1. Look first: Is there a product, quotient, or composition?
  2. Outer structure: What's the main operation connecting parts?
  3. Inner structure: Do any parts need the chain rule?
  4. Apply systematically: Handle the outer operation first
  5. Simplify: Expand and collect like terms at the end

Product + Chain

Example: $y = x^2(3x + 1)^4$

First: Product rule
Then: Chain rule on second part

Quotient + Chain

Example: $y = \frac{(x + 1)^2}{x}$

First: Quotient rule
Then: Chain rule on top

Chain + Product

Example: $y = (x^2 \cdot 2x)^3$

First: Chain rule (outer)
Then: Product rule (inside)

Worked Examples

Example 1: Product Rule + Chain Rule

Product Rule Chain Rule
Differentiate: $y = x^2(2x + 1)^3$
Step 1: Identify structure ▼
Product of $u = x^2$ and $v = (2x + 1)^3$
Step 2: Find $u'$ ▼
$u' = 2x$ (simple power rule)
Step 3: Find $v'$ (chain rule needed!) ▼
Let $w = 2x + 1$
$v' = 3w^2 \times 2 = 6(2x + 1)^2$
Step 4: Apply product rule ▼
$y' = uv' + vu'$
$y' = x^2 \cdot 6(2x + 1)^2 + (2x + 1)^3 \cdot 2x$
✓ Answer: Factor ▼
$y' = 2x(2x + 1)^2[3x + (2x + 1)]$
$y' = 2x(2x + 1)^2(5x + 1)$

Example 2: Quotient Rule + Chain Rule

Quotient Rule Chain Rule
Differentiate: $y = \frac{(x^2 + 1)^2}{x}$
Step 1: Identify ▼
$u = (x^2 + 1)^2$ (top), $v = x$ (bottom)
Step 2: Find $u'$ (chain rule) ▼
$u' = 2(x^2 + 1) \times 2x = 4x(x^2 + 1)$
Step 3: Find $v'$ ▼
$v' = 1$
Step 4: Apply quotient rule ▼
$y' = \frac{vu' - uv'}{v^2}$
$y' = \frac{x \cdot 4x(x^2 + 1) - (x^2 + 1)^2 \cdot 1}{x^2}$
✓ Answer: Simplify ▼
$y' = \frac{4x^2(x^2 + 1) - (x^2 + 1)^2}{x^2}$
$y' = \frac{(x^2 + 1)[4x^2 - (x^2 + 1)]}{x^2} = \frac{(x^2 + 1)(3x^2 - 1)}{x^2}$

Example 3: Chain on Product

Chain Rule Product Rule
Differentiate: $y = [x(x + 1)]^2$
Step 1: Identify outer function ▼
Something squared, so use chain rule first
Let $u = x(x + 1)$, so $y = u^2$
Step 2: Find $\frac{dy}{du}$ ▼
$\frac{dy}{du} = 2u$
Step 3: Find $\frac{du}{dx}$ (product rule) ▼
$\frac{du}{dx} = x \cdot 1 + (x + 1) \cdot 1 = x + x + 1 = 2x + 1$
Step 4: Chain rule ▼
$\frac{dy}{dx} = 2u \times (2x + 1) = 2[x(x + 1)](2x + 1)$
✓ Answer: Simplify ▼
$\frac{dy}{dx} = 2x(x + 1)(2x + 1)$

Practice: Product + Chain Rule

Set 1: Varying the power in the second function

First function is $x^2$, second function $(x + 1)^n$ varies

a) $y = x^2(x + 1)^2$
$u = x^2, u' = 2x$
$v = (x + 1)^2, v' = 2(x + 1)$
$y' = x^2 \cdot 2(x + 1) + (x + 1)^2 \cdot 2x$
$y' = 2x(x + 1)[x + (x + 1)] = 2x(x + 1)(2x + 1)$
b) $y = x^2(x + 1)^3$
$u = x^2, u' = 2x$
$v = (x + 1)^3, v' = 3(x + 1)^2$
$y' = x^2 \cdot 3(x + 1)^2 + (x + 1)^3 \cdot 2x$
$y' = x(x + 1)^2[3x + 2(x + 1)] = x(x + 1)^2(5x + 2)$
c) $y = x^2(x + 1)^4$
$u = x^2, u' = 2x$
$v = (x + 1)^4, v' = 4(x + 1)^3$
$y' = x^2 \cdot 4(x + 1)^3 + (x + 1)^4 \cdot 2x$
$y' = 2x(x + 1)^3[2x + (x + 1)] = 2x(x + 1)^3(3x + 1)$

Set 2: Varying the coefficient inside brackets

Pattern: $x(ax + 1)^2$ with different values of $a$

a) $y = x(x + 1)^2$
$u = x, u' = 1; v = (x + 1)^2, v' = 2(x + 1)$
$y' = x \cdot 2(x + 1) + (x + 1)^2 \cdot 1$
$y' = (x + 1)[2x + (x + 1)] = (x + 1)(3x + 1)$
b) $y = x(2x + 1)^2$
$u = x, u' = 1; v = (2x + 1)^2, v' = 4(2x + 1)$
$y' = x \cdot 4(2x + 1) + (2x + 1)^2 \cdot 1$
$y' = (2x + 1)[4x + (2x + 1)] = (2x + 1)(6x + 1)$
c) $y = x(3x + 1)^2$
$u = x, u' = 1; v = (3x + 1)^2, v' = 6(3x + 1)$
$y' = x \cdot 6(3x + 1) + (3x + 1)^2 \cdot 1$
$y' = (3x + 1)[6x + (3x + 1)] = (3x + 1)(9x + 1)$

Practice: Quotient + Chain Rule

Set 3: Varying the power in numerator

Pattern: $\frac{(x + 1)^n}{x}$

a) $y = \frac{(x + 1)^2}{x}$
$u = (x + 1)^2, u' = 2(x + 1); v = x, v' = 1$
$y' = \frac{x \cdot 2(x + 1) - (x + 1)^2 \cdot 1}{x^2}$
$y' = \frac{(x + 1)[2x - (x + 1)]}{x^2} = \frac{(x + 1)(x - 1)}{x^2}$
b) $y = \frac{(x + 1)^3}{x}$
$u = (x + 1)^3, u' = 3(x + 1)^2; v = x, v' = 1$
$y' = \frac{x \cdot 3(x + 1)^2 - (x + 1)^3 \cdot 1}{x^2}$
$y' = \frac{(x + 1)^2[3x - (x + 1)]}{x^2} = \frac{(x + 1)^2(2x - 1)}{x^2}$
c) $y = \frac{(x + 1)^4}{x}$
$u = (x + 1)^4, u' = 4(x + 1)^3; v = x, v' = 1$
$y' = \frac{x \cdot 4(x + 1)^3 - (x + 1)^4 \cdot 1}{x^2}$
$y' = \frac{(x + 1)^3[4x - (x + 1)]}{x^2} = \frac{(x + 1)^3(3x - 1)}{x^2}$

Set 4: Challenge problems

a) $y = [x^2(x + 1)]^2$
Chain rule first: $u = x^2(x + 1), y = u^2$
$\frac{du}{dx} = x^2 \cdot 1 + (x + 1) \cdot 2x = x^2 + 2x^2 + 2x = 3x^2 + 2x$
$\frac{dy}{dx} = 2u \cdot (3x^2 + 2x) = 2x^2(x + 1)(3x^2 + 2x)$
$= 2x(x + 1)(3x^2 + 2x) \cdot x = 2x^2(x + 1)(3x + 2)$
b) $y = \frac{x^2}{(x + 1)^2}$
$u = x^2, u' = 2x; v = (x + 1)^2, v' = 2(x + 1)$
$y' = \frac{(x + 1)^2 \cdot 2x - x^2 \cdot 2(x + 1)}{(x + 1)^4}$
$y' = \frac{2x(x + 1)[(x + 1) - x]}{(x + 1)^4} = \frac{2x}{(x + 1)^3}$

Key Strategy Summary

  • ✓ Identify the structure: What's the main operation?
  • ✓ Work from outside in: Handle outer operations first
  • ✓ Chain rule often appears: Especially in composite functions
  • ✓ Factor common terms: Makes simplification much easier
  • ✓ Practice recognition: Pattern recognition speeds up solutions
