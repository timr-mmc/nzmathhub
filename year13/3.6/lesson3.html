Differentiating Composite Functions

When one function is inside another (like $(x^2 + 1)^3$), we need the chain rule. Think of it as peeling an onion - differentiate the outer layer, then multiply by the derivative of the inner layer.

The Chain Rule

If $y$ is a function of $u$, and $u$ is a function of $x$, then:

$$\frac{dy}{dx} = \frac{dy}{du} \times \frac{du}{dx}$$

Or in words: "Derivative of outer with respect to inner, times derivative of inner with respect to x"

Identifying Composite Functions

Example: $y = (x^2 + 1)^3$

  • Outer function: $( \ )^3$ — something cubed
  • Inner function: $x^2 + 1$ — what's inside the brackets

We can write: $y = u^3$ where $u = x^2 + 1$

Worked Examples

Example 1: Basic Chain Rule

Differentiate: $y = (x + 1)^2$

1 Set up: Let $u = x + 1$, so $y = u^2$
2 Find $\frac{dy}{du}$: $\frac{dy}{du} = 2u$
3 Find $\frac{du}{dx}$: $\frac{du}{dx} = 1$
4 Apply chain rule: $\frac{dy}{dx} = 2u \times 1 = 2u$
5 Substitute back: $\frac{dy}{dx} = 2(x + 1)$

Example 2: Higher Power

Differentiate: $y = (2x + 3)^4$

1 Set up: Let $u = 2x + 3$, so $y = u^4$
2 Find $\frac{dy}{du}$: $\frac{dy}{du} = 4u^3$
3 Find $\frac{du}{dx}$: $\frac{du}{dx} = 2$
4 Apply chain rule: $\frac{dy}{dx} = 4u^3 \times 2 = 8u^3$
5 Substitute back: $\frac{dy}{dx} = 8(2x + 3)^3$

Example 3: Quadratic Inside

Differentiate: $y = (x^2 + 1)^3$

1 Set up: Let $u = x^2 + 1$, so $y = u^3$
2 Find $\frac{dy}{du}$: $\frac{dy}{du} = 3u^2$
3 Find $\frac{du}{dx}$: $\frac{du}{dx} = 2x$
4 Apply chain rule: $\frac{dy}{dx} = 3u^2 \times 2x = 6xu^2$
5 Substitute back: $\frac{dy}{dx} = 6x(x^2 + 1)^2$

Example 4: Negative Power

Differentiate: $y = (3x - 1)^{-2}$

1 Set up: Let $u = 3x - 1$, so $y = u^{-2}$
2 Find $\frac{dy}{du}$: $\frac{dy}{du} = -2u^{-3}$
3 Find $\frac{du}{dx}$: $\frac{du}{dx} = 3$
4 Apply chain rule: $\frac{dy}{dx} = -2u^{-3} \times 3 = -6u^{-3}$
5 Substitute back: $\frac{dy}{dx} = -6(3x - 1)^{-3} = \frac{-6}{(3x - 1)^3}$

Practice: Minimal Change

Set 1: Changing the power only

Notice: The inner function stays the same, only the power changes

a) $y = (x + 1)^2$
$u = x + 1, \frac{du}{dx} = 1$
$\frac{dy}{du} = 2u$
$\frac{dy}{dx} = 2u \times 1 = 2(x + 1)$
b) $y = (x + 1)^3$
$u = x + 1, \frac{du}{dx} = 1$
$\frac{dy}{du} = 3u^2$
$\frac{dy}{dx} = 3u^2 \times 1 = 3(x + 1)^2$
c) $y = (x + 1)^4$
$u = x + 1, \frac{du}{dx} = 1$
$\frac{dy}{du} = 4u^3$
$\frac{dy}{dx} = 4u^3 \times 1 = 4(x + 1)^3$
d) $y = (x + 1)^5$
$u = x + 1, \frac{du}{dx} = 1$
$\frac{dy}{du} = 5u^4$
$\frac{dy}{dx} = 5u^4 \times 1 = 5(x + 1)^4$

Set 2: Changing the coefficient inside

Notice: Only the coefficient of x changes

a) $y = (x + 1)^3$
$u = x + 1, \frac{du}{dx} = 1$
$\frac{dy}{dx} = 3u^2 \times 1 = 3(x + 1)^2$
b) $y = (2x + 1)^3$
$u = 2x + 1, \frac{du}{dx} = 2$
$\frac{dy}{dx} = 3u^2 \times 2 = 6(2x + 1)^2$
c) $y = (3x + 1)^3$
$u = 3x + 1, \frac{du}{dx} = 3$
$\frac{dy}{dx} = 3u^2 \times 3 = 9(3x + 1)^2$
d) $y = (4x + 1)^3$
$u = 4x + 1, \frac{du}{dx} = 4$
$\frac{dy}{dx} = 3u^2 \times 4 = 12(4x + 1)^2$

Set 3: Quadratic expressions inside

Notice: The power of x inside changes

a) $y = (x + 1)^2$
$u = x + 1, \frac{du}{dx} = 1$
$\frac{dy}{dx} = 2u \times 1 = 2(x + 1)$
b) $y = (x^2 + 1)^2$
$u = x^2 + 1, \frac{du}{dx} = 2x$
$\frac{dy}{dx} = 2u \times 2x = 4x(x^2 + 1)$
c) $y = (x^3 + 1)^2$
$u = x^3 + 1, \frac{du}{dx} = 3x^2$
$\frac{dy}{dx} = 2u \times 3x^2 = 6x^2(x^3 + 1)$
d) $y = (x^4 + 1)^2$
$u = x^4 + 1, \frac{du}{dx} = 4x^3$
$\frac{dy}{dx} = 2u \times 4x^3 = 8x^3(x^4 + 1)$

Set 4: Mixed practice

Apply what you've learned - identify inner and outer functions

a) $y = (5x - 2)^4$
$u = 5x - 2, \frac{du}{dx} = 5$
$\frac{dy}{dx} = 4u^3 \times 5 = 20(5x - 2)^3$
b) $y = (x^2 - 3)^5$
$u = x^2 - 3, \frac{du}{dx} = 2x$
$\frac{dy}{dx} = 5u^4 \times 2x = 10x(x^2 - 3)^4$
c) $y = (2x^2 + 4)^3$
$u = 2x^2 + 4, \frac{du}{dx} = 4x$
$\frac{dy}{dx} = 3u^2 \times 4x = 12x(2x^2 + 4)^2$
d) $y = (3 - x)^{-1}$
$u = 3 - x, \frac{du}{dx} = -1$
$\frac{dy}{dx} = -1u^{-2} \times (-1) = u^{-2} = (3 - x)^{-2} = \frac{1}{(3 - x)^2}$

Key Points to Remember

  • ✓ Chain Rule: $\frac{dy}{dx} = \frac{dy}{du} \times \frac{du}{dx}$
  • ✓ Identify the inner function (what's inside brackets)
  • ✓ Identify the outer function (what's done to the brackets)
  • ✓ Differentiate outer, keep inner unchanged
  • ✓ Multiply by the derivative of the inner function
  • ✓ Substitute back to express in terms of $x$
