Finding Maximum and Minimum Points

A stationary point occurs where the gradient is zero. At these points, the curve is momentarily horizontal. They can be maxima, minima, or points of inflection.

Finding Stationary Points

  1. Find $\frac{dy}{dx}$: Differentiate the function
  2. Set $\frac{dy}{dx} = 0$: Solve for $x$-coordinates
  3. Find $y$-coordinates: Substitute back into original equation
  4. Determine nature: Use second derivative test

Determining the Nature of Stationary Points

Maximum

$\frac{d^2y}{dx^2} < 0$

Negative
Concave down
∩

Minimum

$\frac{d^2y}{dx^2} > 0$

Positive
Concave up
∪

Inflection

$\frac{d^2y}{dx^2} = 0$

Zero
Further testing needed
~

Second Derivative Test Summary

  • Find $\frac{d^2y}{dx^2}$ by differentiating $\frac{dy}{dx}$ again
  • Evaluate $\frac{d^2y}{dx^2}$ at each stationary point
  • If positive → minimum point (smiling curve)
  • If negative → maximum point (frowning curve)
  • If zero → need first derivative test

Worked Examples

Example 1: Basic Quadratic

Question: Find and classify the stationary points of $y = x^2 - 4x + 1$
Step 1: Differentiate the function ▼
$$\frac{dy}{dx} = 2x - 4$$
Step 2: Set the derivative equal to zero ▼
$$2x - 4 = 0$$
$$2x = 4$$
$$x = 2$$
Step 3: Find the y-coordinate ▼
$$y = (2)^2 - 4(2) + 1$$
$$y = 4 - 8 + 1$$
$$y = -3$$
Step 4: Find the second derivative ▼
$$\frac{d^2y}{dx^2} = 2$$
Since $\frac{d^2y}{dx^2} = 2 > 0$, the curve is concave up (smiling)
✓ Answer ▼
Minimum point at $(2, -3)$

Example 2: Cubic Function with Two Stationary Points

Question: Find and classify all stationary points of $y = x^3 - 3x^2 + 4$
Step 1: Differentiate the function ▼
$$\frac{dy}{dx} = 3x^2 - 6x$$
Step 2: Set the derivative equal to zero and solve ▼
$$3x^2 - 6x = 0$$
$$3x(x - 2) = 0$$
$$x = 0 \text{ or } x = 2$$
Step 3: Find y-coordinates for each x-value ▼
At $x = 0$:
$$y = (0)^3 - 3(0)^2 + 4$$
$$y = 4$$
At $x = 2$:
$$y = (2)^3 - 3(2)^2 + 4$$
$$y = 8 - 12 + 4$$
$$y = 0$$
Step 4: Find the second derivative ▼
$$\frac{d^2y}{dx^2} = 6x - 6$$
Step 5: Test each point using the second derivative ▼
At $x = 0$:
$$\frac{d^2y}{dx^2} = 6(0) - 6 = -6$$
Since $-6 < 0$, this is a maximum
At $x = 2$:
$$\frac{d^2y}{dx^2} = 6(2) - 6 = 6$$
Since $6 > 0$, this is a minimum
✓ Answer ▼
Maximum point at $(0, 4)$
Minimum point at $(2, 0)$

Example 3: Factored Form (Using Quadratic Formula)

Question: Find and classify the stationary points of $y = x(x - 2)(x + 1)$
Step 1: Expand the function ▼
$$y = x(x - 2)(x + 1)$$
$$y = x(x^2 - x - 2)$$
$$y = x^3 - x^2 - 2x$$
Step 2: Differentiate the function ▼
$$\frac{dy}{dx} = 3x^2 - 2x - 2$$
Step 3: Set equal to zero and use quadratic formula ▼
$$3x^2 - 2x - 2 = 0$$
$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$
$$x = \frac{2 \pm \sqrt{(-2)^2 - 4(3)(-2)}}{2(3)}$$
$$x = \frac{2 \pm \sqrt{4 + 24}}{6}$$
$$x = \frac{2 \pm \sqrt{28}}{6}$$
$$x = \frac{2 \pm 2\sqrt{7}}{6}$$
$$x = \frac{1 \pm \sqrt{7}}{3}$$
Step 4: Find the second derivative ▼
$$\frac{d^2y}{dx^2} = 6x - 2$$
Step 5: Test each x-value ▼
At $x = \frac{1 + \sqrt{7}}{3} \approx 1.22$:
$$\frac{d^2y}{dx^2} = 6\left(\frac{1 + \sqrt{7}}{3}\right) - 2 > 0$$
Positive, so this is a minimum
At $x = \frac{1 - \sqrt{7}}{3} \approx -0.55$:
$$\frac{d^2y}{dx^2} = 6\left(\frac{1 - \sqrt{7}}{3}\right) - 2 < 0$$
Negative, so this is a maximum
✓ Answer ▼
Maximum at $x = \frac{1 - \sqrt{7}}{3}$
Minimum at $x = \frac{1 + \sqrt{7}}{3}$

Practice: Minimal Change

Set 1: Changing the linear coefficient

Pattern: $y = x^2 - ax + 1$

a) Find stationary points of $y = x^2 - 2x + 1$
$\frac{dy}{dx} = 2x - 2 = 0 \Rightarrow x = 1$
$y = 1 - 2 + 1 = 0$
$\frac{d^2y}{dx^2} = 2 > 0$ → Minimum at $(1, 0)$
b) Find stationary points of $y = x^2 - 4x + 1$
$\frac{dy}{dx} = 2x - 4 = 0 \Rightarrow x = 2$
$y = 4 - 8 + 1 = -3$
$\frac{d^2y}{dx^2} = 2 > 0$ → Minimum at $(2, -3)$
c) Find stationary points of $y = x^2 - 6x + 1$
$\frac{dy}{dx} = 2x - 6 = 0 \Rightarrow x = 3$
$y = 9 - 18 + 1 = -8$
$\frac{d^2y}{dx^2} = 2 > 0$ → Minimum at $(3, -8)$

Set 2: Cubic functions - changing coefficient

Pattern: $y = x^3 - ax^2$

a) Stationary points of $y = x^3 - 3x^2$
$\frac{dy}{dx} = 3x^2 - 6x = 3x(x - 2) = 0$
$x = 0$ or $x = 2$
At $x = 0$: $y = 0$, $\frac{d^2y}{dx^2} = -6 < 0$ → Maximum $(0, 0)$
At $x = 2$: $y = -4$, $\frac{d^2y}{dx^2} = 6 > 0$ → Minimum $(2, -4)$
b) Stationary points of $y = x^3 - 6x^2$
$\frac{dy}{dx} = 3x^2 - 12x = 3x(x - 4) = 0$
$x = 0$ or $x = 4$
At $x = 0$: $y = 0$, $\frac{d^2y}{dx^2} = -12 < 0$ → Maximum $(0, 0)$
At $x = 4$: $y = -32$, $\frac{d^2y}{dx^2} = 12 > 0$ → Minimum $(4, -32)$
c) Stationary points of $y = x^3 - 9x^2$
$\frac{dy}{dx} = 3x^2 - 18x = 3x(x - 6) = 0$
$x = 0$ or $x = 6$
At $x = 0$: $y = 0$, $\frac{d^2y}{dx^2} = -18 < 0$ → Maximum $(0, 0)$
At $x = 6$: $y = -108$, $\frac{d^2y}{dx^2} = 18 > 0$ → Minimum $(6, -108)$

Set 3: Adding linear terms

Pattern: $y = x^3 - 3x^2 + c$

a) Stationary points of $y = x^3 - 3x^2$
$\frac{dy}{dx} = 3x^2 - 6x = 0 \Rightarrow x = 0$ or $x = 2$
Maximum at $(0, 0)$, Minimum at $(2, -4)$
b) Stationary points of $y = x^3 - 3x^2 + 5$
$\frac{dy}{dx} = 3x^2 - 6x = 0 \Rightarrow x = 0$ or $x = 2$
(Same $x$-values, $y$-values shifted up by 5)
Maximum at $(0, 5)$, Minimum at $(2, 1)$
c) Stationary points of $y = x^3 - 3x^2 - 10$
$\frac{dy}{dx} = 3x^2 - 6x = 0 \Rightarrow x = 0$ or $x = 2$
(Same $x$-values, $y$-values shifted down by 10)
Maximum at $(0, -10)$, Minimum at $(2, -14)$

Set 4: Complete analysis

a) Find and classify stationary points of $y = 2x^3 - 9x^2 + 12x$
$\frac{dy}{dx} = 6x^2 - 18x + 12 = 6(x^2 - 3x + 2) = 6(x - 1)(x - 2) = 0$
$x = 1$ or $x = 2$
At $x = 1$: $y = 5$, $\frac{d^2y}{dx^2} = 12(1) - 18 = -6 < 0$ → Max $(1, 5)$
At $x = 2$: $y = 4$, $\frac{d^2y}{dx^2} = 12(2) - 18 = 6 > 0$ → Min $(2, 4)$
b) Find and classify stationary points of $y = x^4 - 8x^2$
$\frac{dy}{dx} = 4x^3 - 16x = 4x(x^2 - 4) = 4x(x - 2)(x + 2) = 0$
$x = 0, 2, -2$
$\frac{d^2y}{dx^2} = 12x^2 - 16$
At $x = 0$: $y = 0$, $\frac{d^2y}{dx^2} = -16 < 0$ → Max $(0, 0)$
At $x = \pm 2$: $y = -16$, $\frac{d^2y}{dx^2} = 32 > 0$ → Min $(\pm 2, -16)$

Key Points to Remember

  • ✓ Stationary points occur where $\frac{dy}{dx} = 0$
  • ✓ Always find both $x$ and $y$ coordinates
  • ✓ Use second derivative to classify: positive → minimum, negative → maximum
  • ✓ Cubic functions typically have one maximum and one minimum
  • ✓ Quadratics have only one stationary point (turning point)
