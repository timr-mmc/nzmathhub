Standard Overview

#
Standard: AS91576 - Critical Path
Credits: 2
Level: NCEA Level 3

Achievement Standard

This standard involves using critical path analysis to optimize models. Students will work with network diagrams and project management techniques.

What You'll Learn

  • Creating and interpreting network diagrams
  • Identifying critical paths in projects
  • Calculating earliest start and finish times
  • Determining float times
  • Optimizing project schedules
