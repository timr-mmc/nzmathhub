4. Additional Features

🌙 Dark Mode

Click the button in the bottom-left corner (or press Ctrl+D) to toggle dark mode. Your preference is saved automatically!

Benefits: Reduces eye strain, better for low-light environments, maintains accessibility standards.

🖨️ Print Worksheets

Click the "Print Worksheet" button in the top-right corner to generate a print-friendly version.

Features: Hides navigation, shows answers by default, professional formatting for A4 paper.

⌨️ Keyboard Shortcuts

Navigate more efficiently with keyboard shortcuts:

← Left Arrow Go to previous page
→ Right Arrow Go to next page
Space Toggle first answer
Ctrl+D Toggle dark mode

5. Traditional Practice Problems

Click "Show Answer" to reveal solutions. Math renders beautifully with KaTeX!

1. Solve for $x$: $2x + 5 = 13$

Working:

$$2x + 5 = 13$$

$$2x = 13 - 5$$

$$2x = 8$$

$$x = 4$$

2. Differentiate $f(x) = x^3 + 2x^2 - 5x + 1$

Using the power rule:

$$f'(x) = 3x^2 + 4x - 5$$

Getting Started

How to Use These Features in Your Lessons

Step 1: Include the required CSS and JavaScript files in your HTML <head>:

<link rel="stylesheet" href="../../interactive.css">
<link rel="stylesheet" href="../../dark-mode.css">
<link rel="stylesheet" href="../../print.css">
<script defer src="../../interactive.js"></script>

Step 2: Choose a template from /templates/ folder

Step 3: Add interactive components using the examples in this demo
